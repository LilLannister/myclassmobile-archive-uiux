# MyClassMobile Archive UI/UX

A modern mobile-first LMS archive application UI/UX project designed in Figma.

This project includes:
- Design System
- Wireframes
- Final Mobile UI
- Reusable Components
- Variants & Auto Layout
- Course archive experience
